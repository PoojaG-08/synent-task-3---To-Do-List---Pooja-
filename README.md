# To-Do List (CLI)

A simple and professional To-Do List application built using Python.  
This CLI-based project helps users manage daily tasks efficiently.

---

## Features

- Add tasks
- Delete tasks
- View tasks
- Menu-driven interface
- User-friendly CLI interaction
- Invalid input handling

---

## Technologies Used

- Python 3

---

## Project Structure

todo-list/

│

├── todo.py

└── README.md

---

## How to Run

1. Open terminal or command prompt

2. Navigate to the project folder

  cd todo-list

3. Run the program

  python todo.py
